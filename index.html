群规 · 当前版本

// 小荷包规则
¥每人先放 50 元进入支付宝小荷包
¥当天没打卡 = -5 元进入公共池
¥每周日结算：第一名拿公共池 60%，第二名拿 30%，第三名拿 10%
¥如果三人都正常完成，则押金沿用到下个月
¥直到谁先达到自己设立的目标，谁可以一次性拿走全部押金
// 补救机制
→每人每周有 3 次补卡机会，可以补 3 天，不扣钱
→整体按“上四休三”理解，允许有休息和缓冲空间
→补卡用于修复中断，不用于制造完美假象
// 特殊情况
01经期不计入运动打卡要求
02如经期当天未发生暴食行为，则算打卡完成
// 最重要的原则
!允许有断掉的，但不能连续断，断开后要及时开始
!最低可以只做 20 分钟
!机制的目标是让三个人长期不断线，而不是制造压力表演
// 技术说明
→所有打卡数据会保存在 GitHub 仓库中
→由后端接口写入 GitHub 数据文件
→争议以仓库记录为准
